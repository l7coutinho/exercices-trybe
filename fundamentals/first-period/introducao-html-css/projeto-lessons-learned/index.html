Comandos Git e GitHub

mkdir: cria uma pasta Git add: adiciona os arquivos ao diretorio
Git init: inicia o git em um repositorio local Git commit -m: Comita informando mensagem
Git branch 'nome': cria uma nova branch Git push: envia o conteudo para o repositorio remoto
Git merge: é o comando que unifica os conteudos das branch's Git pull: atualiza seu repositorio com o conteudo remoto
